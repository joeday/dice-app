# Minimalist Dice
A simple 6-sided die to use for games. I built this when my daughters Frozen pop-up game stopped popping.

## How To Role the Die
First, go to http://joeday.github.io/dice on any browser. Then...

#### On Mobile
Touch the die. 

#### On Desktop
You have two options to roll the die:

1.  Press Spacebar
2.  Click the die
